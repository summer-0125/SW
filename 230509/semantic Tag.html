폰트를 두껍게

<strong>태그는 Semantic Tag로 HTML문서 내에서 강조하고 싶은 내용에 사용한다. 둘 다 폰트가 두꺼워지지만 이건 내용을 강조한다.

<i>태그는 CSS처럼 웹 사이트에 표시되는 모양만 결정한다. 이텔릭체 내용 강조 없이 모양만 이탤릭체로 표현하고 싶을 때는 CSS font-styly 속성을 사용한다.

<em>태그는 Semantic Tag로 HTML에서 이탤릭체로 내용을 강조하고 싶을 때 사용한다.

<s>태그는 화면상에 표시되는 모양만 결정한다. 취소선 취소된 내용이라는 강조가 아닌 디자인적으로 중간에 가로선을 긋고 싶을 때 CSS text-decoration 속성을 사용한다.

<del>태그는 Semantic Tag로 문서 내에서 삭제된 내용이라는 걸 강조하고 싶을 때 사용한다. 이건 Semantic Tag!

<u>태그는 웹 브라우저에 표시되는 모양만 결정한다. 언더라인 CSS text-decoration : Underline

<ins>태그는 Semantic Tag로 HTML 문서에 새로 삽입된 내용이라는 걸 강조하고 싶을 때 사용한다. 새로 삽입된 내용을 강조하는 Semantic Tag!
